# gamuX

linuxgame.cn下一代主题(wordpress)，使用Vuejs

## 体验

```shell
git clone https://github.com/Gamuxorg/gamuX.git
```

将文件夹放在wordpress的wp-content/themes里,在wordpress的后台切换主题即可。

## 贡献

本地修改、调试代码没问题后，pull request即可。
